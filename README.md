# Real-time-browser-tab-usage-analytics-project
# Tab History Analytics

Analyze and visualize your browser tab history to reveal productivity patterns and time usage.

---

## Features

- Load browser tab history CSV data
- Categorize sites by domain and usage type (work, social, entertainment, learning)
- Calculate session durations
- Summarize top sites and usage categories
- Generate visual reports (charts)

---

## Getting Started

1. Place your history CSV file(s) in the project directory.
2. Open and run `Untitled.ipynb` or the main analysis notebook.

---

## Usage

- Update the CSV filename in the notebook/code to match your data.
- Run analysis code to process, categorize, and visualize tab usage.
- Explore summary statistics and generated charts for insights.

---

## File Structure

- `Untitled.ipynb` — Main analysis notebook
- `tab-history-*.csv` — Example browser tab history files
- `README.md` — Project documentation
